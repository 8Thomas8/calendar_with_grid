# CALENDRIER

## TECHNO
Utilisation de :
- HTML
- CSS
- Javascript

## CSS

Utilisation de `grid` pour gérer la disposition des jours et des dates dans le calendrier.

## JAVASCRIPT

Utilisation de Javascript pour générer les dates du calendrier, et les boutons suivant / précédent.
